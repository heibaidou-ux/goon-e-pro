| 分区 | 区域 | 音箱数 | 音源 |

| 3区 | 中茶室A | 1只 | 蓝牙（客人手机配对） |
| 4区 | 中茶室B | 1只 | 蓝牙（客人手机配对） |
| 5区 | 大茶室C | 1只 | 蓝牙（客人手机配对） |

DSP自动切换：茶室默认背景音乐→客人蓝牙配对后自动切蓝牙→断开后恢复背景音乐。
