**动线**：入口 → 照壁（左实右虚）→ 右半门洞 → 展厅（三面墙：左半径墙/照壁背面/大会议室外墙）→ 右转 → 走廊 → 各房间

**沿走廊排列**（近→远）：
- 左手边：大会议室 → 中茶室A → 中茶室B（偏左）→ 大茶室C（偏右，邻工作间）
- 右手边：工作间（沿右半径延伸，无窗）
