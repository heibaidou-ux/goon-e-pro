### 4.2 设备选型优先级

| 优先级 | 设备 | 理由 |
|--------|------|------|
| **P0（一期必装）** | 智能门锁、智能灯光面板、智能家居网关 | 核心服务流程，无替代方案 |
| **P1（一期必装）** | 中央空调控制器、温湿度传感器 | 提升客人体验，降低能耗 |
| **P2（一期选装）** | 电动窗帘、背景音乐音响 | 提升品质，但可通过手动替代 |
| **P3（二期考虑）** | 路由器/AP升级、UPS扩容 | 待一期网络稳定性评估后决定 |

## 5. 安装与施工注意事项

### 5.1 网络基础设施

| 项目 | 要求 |
|------|------|
| **WiFi覆盖** | 全店统一SSID，每个包间信号强度≥-65dBm |
| **VLAN规划** | 划分三个VLAN：客人网络（仅互联网）、设备网络（IoT网关+传感器）、办公网络（店员端+收银） |
| **有线预留** | 每个包间预留至少1根网线（至弱电箱），网关位置预留2根 |
| **供电保障** | 网关、路由器和门锁网关需接入UPS |

### 5.2 安装时序

```
第一轮（装修/改造阶段）：
  ① 布线（网线、零火线、窗帘电源线）
  ② 安装智能灯光面板、电动窗帘导轨
  ③ 安装网关底座、AP底座

第二轮（设备进场阶段）：
  ④ 安装门锁、空调控制器
  ⑤ 安装音响、传感器
  ⑥ 上电调试、网关绑定

第三轮（联调阶段）：
  ⑦ 场景配置（欢迎/退房/K歌/会议）
  ⑧ 离线规则配置
  ⑨ 全流程测试（预约→开门→场景→退房→关设备）
```

### 5.3 关键风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|---------|
| 网关信号覆盖不足 | 门锁/传感器离线 | 施工前进行无线信号勘测，必要时增加网关数量或中继器 |
| 空调协议不兼容 | 无法远程控制 | 施工前确认空调型号与控制器兼容性，备选红外遥控方案 |
| 旧店改造布线困难 | 装修成本增加 | 优先考虑无线方案（Zigbee），减少穿墙布线 |
| 离线密码时钟漂移 | 客人无法开门 | 选择支持NTP自动校时的门锁，定期同步 |
