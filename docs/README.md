## 版本管理规则

| 规则 | 说明 |
|------|------|
| **版本号格式** | V主版本.次版本（如 V1.0, V1.1, V2.0） |
| **主版本递增** | 重大内容重构或架构变更 |
| **次版本递增** | 局部补充、修正、格式调整 |
| **文件名版本** | 文件名中的版本号与文档头版本号一致 |
| **历史保留** | 旧版本文件归档至 `_history/` 子目录，不可覆盖删除 |
